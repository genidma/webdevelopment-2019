Alright, first we are going to categorize items in an ordered list and an un-ordered list.

Ordered list

Ordered lists make use of the ol attribute, the li attribute is embedded within. Here is a sample output:
  1. List item #1
  2. List item #2
  3. List item #3


Rather simple. Do you agree?

Un-ordered list

And now, the unordered list. Here you make use of the ul tag and as illustrated above, you then embed the li tag within. Here is a sample output:
  • Sample task # 1
  • Sample task # 2
  • Sample task # 3
    1. Sample task # 3.a
    2. Sample task # 3.b
